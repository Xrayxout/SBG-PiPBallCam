# SBG-PiPBallCam
#### _A BepInEx5 mod to Add a Picture in Picture (PiP) camerea that tracks your golf ball_

# TO-DO

## Installation

- [Install BepInEx 5](https://github.com/BepInEx/BepInEx/releases) 
- Download and drop the SBG-PiPBallCam.dll file in the \BepInEx\plugins folder of your Super Battle Golf Installation


## Building for source
*set the env variable ``SUPER_BATTLE_GOLF_PATH`` to your Super Battle Golf installation directory.*
```sh
dotnet build
```
